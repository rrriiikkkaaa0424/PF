### テーマを選んだ理由
WBCがちょうど活気づいていたときに見かけたニュースで、「大谷翔平とガブリアスの共通点が多く話題」
というニュースを見かけました。そのときに、異なる種族でも共通点が多いのは面白いと思いました。
私も、共通のポケモンは何だろうとサイトを探した結果、見やすいサイトもないし他にはこういうものがあれば良いのにな。と
思ったものも無かったので、自分で作ってしまえば良いんだと思ったことがきっかけでした。
